# **Ejercicio .NET 8 para Hipódromo de Palermo**

## Desplegar la aplicación

1. La solución debe abrise en Visual Studio 2022 como mínimo con .NET 8 instalado
2. El script Reservas.sql debe ejecutarse en MS SQL Server
3. En caso de modificación la cadena de conexión se encuentra en appsettings.json

`"ConnectionStrings": {
  "DefaultConnection": "Server=(localdb)\\mssqllocaldb;Database=Tucson;Trusted_Connection=True;MultipleActiveResultSets=true"
}`

4. Inicializar el proyecto
